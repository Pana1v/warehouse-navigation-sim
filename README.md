# Warehouse Simulation

## Environment Decision
We started by trying the AWS Robomaker Small Warehouse world. It turned out to be too heavy and slow to load for our needs, so we switched to a custom, lightweight warehouse environment in Gazebo Harmonic. This gave us much better performance and reliability.
